# AI Parking Monitoring Lab

## Overview
This project is designed to implement an AI-based system for monitoring parking spaces in real-time using computer vision technology. It is aimed at enhancing the management of parking resources in urban environments, thus reducing congestion and increasing the efficiency of parking.

## Features
- Real-time parking space detection and monitoring.
- User-friendly interface for accessing parking data.
- API integration for third-party services.
- Notification system for available parking spaces.

## Installation
To set up the project locally, follow these steps:
1. Clone the repository:
   ```bash
   git clone https://github.com/aryanj108/aiParkingMonitoringLab.git
   ```
2. Navigate to the project directory:
   ```bash
   cd aiParkingMonitoringLab
   ```
3. Install the required dependencies:
   ```bash
   pip install -r requirements.txt
   ```

## Usage
1. Start the application:
   ```bash
   python app.py
   ```
2. Access the application through your web browser at `http://localhost:5000`.
3. Follow the on-screen instructions to monitor parking spaces.
